## <img src="https://media.giphy.com/media/LnQjpWaON8nhr21vNW/giphy.gif" width="28"> Quick Setup

```bash
# 1️⃣  Clone the repository
git clone https://github.com/SARDAR-RDX/rdx-mega-bot.git
cd rdx-mega-bot

# 2️⃣  Install all dependencies
npm install

# 3️⃣  Add your Facebook cookies to:
#     bot_connect/cookies.txt   (for cookie login)
#     bot_connect/appstate.json (for appstate login)

# 4️⃣  Configure the bot
#     Edit config.json with your settings

# 5️⃣  Start the bot!
npm start
```
